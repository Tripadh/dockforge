# Documentation Images

Place your screenshots and diagrams here:

- `dashboard.png` - Web dashboard screenshot
- `execution.png` - Code execution example
- `architecture.png` - Architecture diagram
- `architecture-visual.png` - Visual container diagram
- `demo.gif` - Animated demo (optional)

## How to Capture Screenshots

1. Start the application: `docker-compose up -d`
2. Open `http://localhost:8080`
3. Write sample code and execute
4. Capture screenshots
5. Save them in this folder
6. Update README.md image paths if needed
